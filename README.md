# WikiPlus

A Desktop client for Wikipedia

## Getting Started

Go to this [Website](http://www.mediafire.com/folder/j0oo616dvoznt/WikiPlus) to download the Latest release

### Prerequisites

```
Java 8
```

### Installing

Download latest release and run it with
```
java -jar WikiPlus.jar
```

## Built With

* JavaFX
